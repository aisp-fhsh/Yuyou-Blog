### Solution
題目給了這張圖片
![](image/題目12(1).png)
因此透過之前的 nmap 掃描後，我發現只有在 port 54321 中才有 robots.txt
訪問後顯示以下內容
```
User-agent: cdxuser
Disallow: /secret-path
```
因此就將 request 的 user-agent 改成 cdxuser ，就可以存取 /secret-path 的內容了
